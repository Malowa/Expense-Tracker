# Python Expense Tracker

A simple GUI expense tracker built with Python.

## Features

- Add expenses
- View expense history
- Categorize spending
- Expense summary reports

## Installation

Clone the repository

git clone https://github.com/Malowa/expense-tracker-python.git

Navigate to project folder

cd expense-tracker-python

Install dependencies

pip install -r requirements.txt

## Usage

Run the application

python main.py
